# Kriya OS

> *Kriya (Sanskrit: क्रिया) — completed action, execution, effort.*

Kriya is an agentic personal operating system designed to autonomously execute tasks, manage workflows, and operate as a digital extension of the self. It favors minimal friction, raw utility, and unpretentious execution.

## Overview

Kriya OS serves as the underlying engine (the yantra) that coordinates various AI agents to perform tasks on your behalf. It is built to be local-first, CLI-native, and highly extensible.
